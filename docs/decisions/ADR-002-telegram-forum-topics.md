# ADR-002: Telegram Forum Supergroup с Topics как UI

**Статус:** Принято  
**Дата:** 2026-04

## Контекст

Нужно отображать сообщения из ~10–50 MAX-чатов в Telegram.

## Варианты

| Вариант | Удобство | Захламление | Ответ |
|---------|----------|------------|-------|
| Forum Supergroup + Topics | ★★★★★ | Изолировано | Reply = ответ в MAX |
| Отдельные группы/каналы | ★★★☆☆ | Хаос в чат-листе | Сложнее |
| Один flat чат с префиксами | ★★☆☆☆ | Каша | Простой reply |

## Решение

**Один Forum Supergroup, каждый MAX-чат = отдельный топик.**

## Обоснование

- Каждый чат изолирован, не мешает другим
- Reply-механика Telegram = натуральный способ ответить
- До ~100 топиков — комфортно для личного использования
- Топик создаётся автоматически при первом сообщении из нового чата

## Последствия

- Нужен Forum Supergroup (не обычная группа)
- Topics должны быть включены в настройках группы
- Бот должен быть администратором с правом управления топиками
